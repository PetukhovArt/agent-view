# Current Stage

## Iteration 1: Core — COMPLETED (2026-03-29)

Commands implemented: `init`, `discover`, `dom`, `stop`

### What works
- `agent-view init` — auto-detects runtime/engine/launch from package.json
- `agent-view discover` — lists CDP targets (windows) as JSON
- `agent-view dom` — accessibility tree in compact text with session refs
- `agent-view dom --filter/--depth/--window` — filtering, depth limit, window selection by ID or title
- `agent-view stop` — stops lazy server
- Lazy server (TCP localhost:47922) — auto-start on first CLI call, 5min idle shutdown
- Tested on `D:\web-projects\web-client` (Electron + Vue 3)

### Integration fixes applied
- IPv4 forced for chrome-remote-interface (Node.js defaults to IPv6)
- Delimiter-based IPC protocol instead of TCP half-close
- Server spawn uses agent-view project root as cwd

### Known issues
- StaticText/InlineTextBox nodes with names not filtered — adds noise, consider aggressive filtering in v2
- Server auto-start via CLI not yet verified (tested with manual server start + existing server)
